## [PRESENTATION EN FRANCAIS]
Ce petit projet consiste à créer un menu interactif en Python en utilisant Pygame et Tkinter pour le jeu Colt Express, un jeu de plateau où les joueurs contrôlent des bandits devant dérober un butin à bord d'un train.   
Pour naviguer à travers les menus, il suffit de cliquer sur les boutons intégrés dans le décor.

> [!TIP]
> Pour lancer le programme:
> ```bash
> python3 projet_interfaceVFinal.py  
> ````
## [ENGLISH PRESENTATION]
This small project consists of creating an interactive menu in Python using Pygame and Tkinter for the game Colt Express, a board game where players control bandits who must steal loot from a train.
To navigate through the menus, simply click on the buttons integrated into the decor.

> [!TIP]
> To start the program:
> ```bash
> python3 project_interfaceVFinal.py
> ````
